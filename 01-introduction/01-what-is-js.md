# JavaScript Kya Hai?

JavaScript ek **programming language** hai jo primarily **web pages ko interactive banane** ke liye use hoti hai.  

### Key Points:
- JS **client-side** aur **server-side** dono pe kaam karta hai (Node.js ke saath).
- JS ko **browser** run karta hai.
- JS se aap web pages mein animations, form validations, event handling aur API calls kar sakte ho.

### Example Use Cases:
1. Button click pe alert show karna  
2. Form validation before submit  
3. Dynamic content update without page reload
