# JavaScript Setup

JavaScript ko run karne ke liye aapko **kisi compiler ki zarurat nahi**, sirf browser ya VS Code chahiye.

### Option 1: Browser Console
1. Browser kholen → F12 → Console tab  
2. Type kare: `console.log("Hello JavaScript");`  
3. Enter → Output dekhen

### Option 2: VS Code + Live Server
1. VS Code install kare  
2. `index.html` file banaye  
3. `<script src="script.js"></script>` add kare  
4. `script.js` file mein JS likhe aur live server run kare
