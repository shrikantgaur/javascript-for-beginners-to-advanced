# Variables in JavaScript

Variables ek container hote hai jisme aap **data store** karte ho.

### Types:
- `var` → Old way, globally/function scoped
- `let` → Modern way, block scoped
- `const` → Constant, block scoped

```js
let name = "Lucknow";   // variable
const age = 25;           // constant
var city = "Delhi";       // old style

###Hinglish Tip:
let aur const prefer kare, var avoid kare.
