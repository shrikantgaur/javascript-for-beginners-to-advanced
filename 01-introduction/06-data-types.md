# Data Types in JavaScript

JavaScript me data types do type ke hote hai:

## 1. Primitive Types
- String → `"Hello"`
- Number → `25`
- Boolean → `true/false`
- Undefined → `let a;`
- Null → `let b = null;`
- Symbol → `Symbol('id')`
- BigInt → `123n`

## 2. Non-Primitive
- Object → `{name: "Shri Kant", age: 25}`
- Array → `[1,2,3,4]`
