# JavaScript Ka History

JavaScript ek aisi language hai jo **1995 se web development ka backbone** bani hui hai. Yahan iska detailed history hai:

- **1995:** Brendan Eich ne Netscape ke liye JavaScript create kiya. Pehla naam tha **Mocha**, fir **LiveScript**, aur finally **JavaScript**.  
- **1996:** Microsoft ne JavaScript adopt kiya aur **JScript** ke naam se Internet Explorer me introduce kiya.  
- **1997:** **ECMAScript 1 (ES1)** release hua, jo JavaScript ka first official standard tha.  
- **1998:** **ECMAScript 2 (ES2)** release hua, minor corrections ke saath.  
- **1999:** **ECMAScript 3 (ES3)** release hua, jo widely adopt hua aur modern JS features ka base ban gaya.  
- **2000–2005:** JavaScript ko zyadatar client-side scripting ke liye use kiya gaya. AJAX (Asynchronous JavaScript and XML) popular hua, jisse web pages dynamically update hone lage without page reload.  
- **2009:** **ECMAScript 5 (ES5)** release hua. Ye version `strict mode`, JSON support aur better array methods ke liye famous hai.  
- **2015:** **ECMAScript 6 (ES6/ES2015)** release hua. ES6 ne modern JS ka foundation banaya: `let`, `const`, arrow functions, classes, modules, template literals aur promises.  
- **2016–2023:** ECMAScript har saal update hota raha (ES2016, ES2017, … ES2023) with new features like async/await, optional chaining, BigInt, nullish coalescing, and more.  
- **Today:** JavaScript **full-stack language** ban chuka hai — browser, server (Node.js), mobile apps (React Native), desktop apps (Electron), aur even machine learning me use hota hai.
